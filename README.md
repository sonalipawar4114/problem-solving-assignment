# Case Study: Reducing Food Waste in School Cafeteria

This repository contains a case study analyzing the problem of **food waste in school cafeterias**, exploring multiple solutions, evaluating them, and recommending an effective strategy.

## PDF Version

The full case study is available in PDF format:  
[View/Download Case Study](Case-Study.pdf)

## Overview

1. **Introduction**  
   - Highlights the significance of food waste and its impact on students, staff, administration, and the environment.

2. **Problem Analysis**  
   - Identifies root causes such as large portions, overestimation of food intake, lack of awareness, and inefficient meal planning.

3. **Possible Solutions**  
   - Adjust portion sizes  
   - Implement “Take-What-You-Can-Eat” policy  
   - Awareness campaigns  
   - Food donation program  
   - Digital feedback system  

4. **Evaluation**  
   - Pros, cons, feasibility, and impact of each solution are analyzed in a table format.

5. **Recommendation**  
   - Combined approach: smaller portions, self-service, and awareness campaigns for maximum effectiveness.

6. **Conclusion**  
   - Emphasizes sustainable problem-solving and long-term behavioral change.
